## Description

A simple JSON editor.

## Installation

There's `.nvmrc` file with the required node version.

To install packages:
```shell
npm i
```

## Dev build

To start dev server:
```shell
npm run dev
```

The server would be accessible on:
```
http://localhost:5173
```

## Release build

To build the application:
```shell
npm run build
```

To start prod preview server:
```shell
npm run preview
```

The server would be accessible on:
```
http://localhost:4173
```
